# AutoSkip

Autoskipping plugin for Lampa: detects intro/credits in any video and shows a 'Skip' button with auto-timers.

Use the script from `plugins/plugin.js` or load directly:

- `https://raw.githubusercontent.com/iLibrium/lampa-plugins/main/plugins/plugin.js`
- `https://cdn.jsdelivr.net/gh/iLibrium/lampa-plugins@main/plugins/plugin.js`

## Development

`plugins/plugin.js` is generated from sources in `src/`.

- `npm install`
- `npm run build`
- `npm run watch`
